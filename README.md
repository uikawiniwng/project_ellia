# Project Ellia

命定系统「艾莉亚核心」的前端仓库

这个仓库目前主要维护两块可直接构建的界面模块：

| 模块 | 说明 | 入口文档 |
| --- | --- | --- |
| `dialog_beauty` | 对话美化卡片，负责把消息里的 `<ellia>` 区块渲染成可播放、可设定的艾莉亚对话卡 | [src/dialog_beauty/README.md](src/dialog_beauty/README.md) |
| `ticket_skill` | 票券式技能展示与导入界面，偏向技能说明、纪念册与设定页呈现 | [src/ticket_skill/README.md](src/ticket_skill/README.md) |
